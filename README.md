# PolarityBot

Discord Bot Written in Python.


# Help
  If you need to login with token, go to https://discordapp.com/developers and find your app, add a bot user if there is none, and      
  copy/paste the token.
  
  If you need a Discord OAUTH Link, use the link down below, and replace CLIENT_ID with the Client ID in your app details. 
  https://discordapp.com/oauth2/authorize?&client_id=CLIENT_ID&scope=bot&permissions=administrator&response_type=code
