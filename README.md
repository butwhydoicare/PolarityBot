# PolarityBot

Discord Bot Written in Python.


# Help
  If you still need to login with token, go to https://discordapp.com/developers and find your app, add a bot user if there is none, and   
  copy/paste the token.
  
  If you need a Discord OAUTH Link, use the link down below, and replace CLIENT_ID with the Client ID in your app details. 
  https://discordapp.com/oauth2/authorize?&client_id=CLIENT_ID&scope=bot&permissions=administrator&response_type=code



# Instructions
  
  1. Install Python: https://www.python.org/downloads/release/python-360/
  2. Install required modules by going into the directory you installed PolarityBot in, then typing ***python -m pip install -r requirements.txt***
  3. Install FFMPEG : http://adaptivesamples.com/how-to-install-ffmpeg-on-windows/
  4. Set up your bot at https://discordapp.com/developers 
  5. you need to login with token, go to https://discordapp.com/developers and find your app, add a bot user if there is none, and      
     copy/paste the token. 
     
     If you need a Discord OAUTH Link, use the link down below, and replace CLIENT_ID with the Client ID in your app details.                  https://discordapp.com/oauth2/authorize?&client_id=CLIENT_ID&scope=bot&permissions=administrator&response_type=code
  
  7. Run your bot.
  8. Have fun!
